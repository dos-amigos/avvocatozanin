Create `site/snippets/breadcrumb.php` — a reusable breadcrumb snippet for all internal pages (chi-sono, servizi/*, come-lavoro, contatti, etc.). It must NOT render on the homepage.

**Semantic requirements (per NAV-04):**
- Outer element: `<nav aria-label="Breadcrumb">`
- List: `<ol class="breadcrumb">` (ordered list — semantically correct for a trail)
- Each step: `<li class="breadcrumb__item">` with an anchor except the last (current page)
- Current page: `<li class="breadcrumb__item breadcrumb__item--active" aria-current="page">` — no anchor
- Separator: `<span class="breadcrumb__separator" aria-hidden="true">/</span>` between items

**Kirby data:**
- Home item always first: link to `$site->url()` with label "Home"
- For pages with depth > 1 (e.g., servizi/diritto-di-famiglia): iterate `$page->parents()->flip()` for intermediate ancestors
- Current page: `$page->title()` as the final non-linked item

**schema.org JSON-LD block (inline `<script type="application/ld+json">`):**
```json
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    { "@type": "ListItem", "position": 1, "name": "Home", "item": "https://..." },
    { "@type": "ListItem", "position": 2, "name": "Servizi", "item": "https://..." },
    { "@type": "ListItem", "position": 3, "name": "Diritto di Famiglia" }
  ]
}
```
Build this dynamically in PHP using a counter and `$page->parents()->flip()` + current page.

**Full snippet structure:**
```php
<?php
// Only render on non-home pages
if ($page->isHomePage()) return;

// Build breadcrumb items: [Home] + ancestors + [current page]
$items = [];
$position = 1;

// Home
$items[] = [
  'name'     => 'Home',
  'url'      => $site->url(),
  'position' => $position++,
  'current'  => false,
];

// Ancestors (e.g., "Servizi" when on a service sub-page)
foreach ($page->parents()->flip() as $ancestor) {
  $items[] = [
    'name'     => $ancestor->title()->html(),
    'url'      => $ancestor->url(),
    'position' => $position++,
    'current'  => false,
  ];
}

// Current page
$items[] = [
  'name'     => $page->title()->html(),
  'url'      => $page->url(),
  'position' => $position,
  'current'  => true,
];
?>

<nav aria-label="Breadcrumb" class="breadcrumb-nav">
  <ol class="breadcrumb">
    <?php foreach ($items as $i => $item): ?>
      <li class="breadcrumb__item<?= $item['current'] ? ' breadcrumb__item--active' : '' ?>"<?= $item['current'] ? ' aria-current="page"' : '' ?>>
        <?php if (!$item['current']): ?>
          <a href="<?= $item['url'] ?>" class="breadcrumb__link"><?= $item['name'] ?></a>
          <span class="breadcrumb__separator" aria-hidden="true">/</span>
        <?php else: ?>
          <?= $item['name'] ?>
        <?php endif ?>
      </li>
    <?php endforeach ?>
  </ol>
</nav>

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    <?php foreach ($items as $i => $item): ?>
    {
      "@type": "ListItem",
      "position": <?= $item['position'] ?>,
      "name": "<?= $item['name'] ?>"
      <?= !$item['current'] ? ', "item": "' . $item['url'] . '"' : '' ?>
    }<?= $i < count($items) - 1 ? ',' : '' ?>
    <?php endforeach ?>
```
