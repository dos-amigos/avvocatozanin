Never center paragraph text (text-align: center) on content sections. Blocks can be centered (margin-inline: auto) but text inside must be left-aligned (bandiera sinistra).

**Why:** User finds centered body text unprofessional and hard to read. Repeated corrections on come-lavoro intro and credentials section.
**How to apply:** When creating any content section, use `text-align: left` by default. Only center headings/subtitles when they're standalone labels, never body paragraphs.
