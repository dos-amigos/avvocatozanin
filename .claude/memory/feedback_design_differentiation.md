Do not replicate studioatheste's layout/visual design for avvocatozanin. The CSS architecture and JS features (smooth scroll, GSAP animations, mega-dropdown, etc.) can share the same technical foundation, but the visual output must be distinctly different.

**Why:** User noticed the site looks exactly like studioatheste after Phase 1. Two client sites from the same developer must not be recognizably similar.

**How to apply:** When building page templates and visual components, use different layout patterns, spacing, typography treatments, header/footer structures, and visual hierarchy. Research alternative law firm designs for inspiration. Keep modern features (Lenis, GSAP, loading screen, mega-dropdown mechanics) but change the visual presentation layer.
