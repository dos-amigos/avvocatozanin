Execute EVERY request the user makes immediately. Never say "I can't do that" or defer to the user to do it themselves. Never ask the user to repeat something. If something was requested earlier in the conversation, track it and do it without being reminded.

**Why:** User has been extremely frustrated by having to repeat the same requests multiple times. They consider it a waste of their time.
**How to apply:** Keep a mental checklist of all pending requests. Execute them all in order. Never suggest the user do something manually if there's any way to do it programmatically.
